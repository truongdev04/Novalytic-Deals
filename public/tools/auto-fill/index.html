Quy trình 4 bước

1

Dán hoặc upload sheet

Copy vùng dữ liệu từ Google Sheets, hoặc tải file .xlsx/.csv lên thẳng.

2

Tool tự chuẩn hóa

Tách store/coupon, nhận diện loại discount (%, $, Free Ship, Deal), gắn cờ dòng thiếu dữ liệu để review.

3

AI viết Description + About

Chọn provider bạn có sẵn key (OpenAI, Anthropic Claude, Gemini, OpenRouter) để sinh nội dung riêng cho từng store.

4

Tải Excel sạch

File gồm 3 sheet Stores · Coupons · Review, import thẳng vào admin NovalyticDeals.

Xem danh sách model

Chỉ dòng có provider đã lưu API key mới được dùng; dòng trùng lựa chọn ở trên bị bỏ qua tự động. Gemini free tính quota riêng theo từng model (đổi model né limit khá hiệu quả); OpenRouter free (":free") giới hạn chủ yếu theo tài khoản/key, nên đổi qua lại các model ":free" trong cùng OpenRouter chưa chắc né được limit — vẫn hữu ích như một danh tính fallback riêng biệt.
